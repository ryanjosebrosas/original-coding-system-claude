# Commands Index

All available slash commands.

## Commands

- `/agents` - Create new subagent
- `/code-review` - Technical code review
- `/code-review-fix` - Fix review issues
- `/commit` - Create git commit
- `/create-pr` - Create GitHub PR
- `/create-prd` - Generate PRD
- `/end-to-end-feature` - Autonomous feature
- `/execute` - Execute plan
- `/execution-report` - Generate report
- `/implement-fix` - Implement fix
- `/init-c` - Initialize project
- `/planning` - Create feature plan
- `/prime` - Prime agent
- `/rca` - Root cause analysis
- `/system-review` - System review
