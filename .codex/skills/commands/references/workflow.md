# Commands Reference

All available slash commands in this project.

## Command List

See .claude/commands/ directory for all command definitions.

## Usage

Commands are invoked with / prefix:
- /prime - Load project context
- /planning [feature] - Create feature plan
- /execute [plan-file] - Execute plan
- /code-review [files] - Review code
- /commit [files] - Create commit
- /create-pr - Create GitHub PR

## Structure

Each command is defined in .claude/commands/[name].md
